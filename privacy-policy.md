# Privacy Policy for Discipline-Chan / Time Tracker Assistant

Last Updated: 17.02.2026

1. Overview

Discipline-Chan (Time Tracker Assistant) is a browser extension designed to help users track time spent on websites and encourage productivity.

We respect your privacy. This extension does not collect, transmit, or sell personal data.

2. Data Collection

This extension:

- Tracks the domain of the active tab and the time spent on it locally in your browser.
- Stores user preferences locally using Chrome storage.
- Does not collect personally identifiable information.
- Does not transmit browsing data to external servers.
- Does not use analytics or tracking services.
- All data remains on your device.

The extension uses browsing activity solely to calculate time spent on websites for the user-facing productivity features. Browsing data is not used for advertising, profiling, analytics, or any secondary purpose. Users may delete all stored data at any time by clearing the extension’s storage or uninstalling the extension.

3. Permissions Usage

The extension uses the following permissions:

- tabs — to detect the active website and categorize time usage
- storage — to store timers, streaks, and settings locally
- host permissions (<all_urls>) — to determine which site you are currently visiting for time tracking purposes
- These permissions are used solely to provide the extension’s core functionality.

The extension does not read, store, or analyze page content, form inputs, passwords, or personal information from websites.

4. Data Sharing

- No data is shared with third parties.
- No data is sold.
- No data is transmitted outside your browser.

5. Security

All information is stored locally using the browser’s built-in storage APIs.

6. Changes to This Policy

If the extension’s functionality changes, this policy will be updated accordingly.
